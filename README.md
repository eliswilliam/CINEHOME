# 🎬 CINEHOME

Application de streaming de films avec interface moderne et backend robuste.

## 📁 Structure du Projet

```
CINEHOME/
├── frontend/          # Interface utilisateur (HTML, CSS, JavaScript)
├── backend-api/       # API REST avec Node.js/Express
├── docs/             # Documentation du projet
├── tests/            # Tests et scripts de test
└── scripts/          # Scripts utilitaires
```

## 🚀 Démarrage Rapide

### Backend
```bash
cd backend-api
npm install
npm start
```

### Frontend
Ouvrez `frontend/index.html` dans votre navigateur ou utilisez un serveur local.

## 🛠️ Technologies

**Frontend:**
- HTML5, CSS3, JavaScript
- API TMDB pour les données de films
- Design responsive

**Backend:**
- Node.js + Express
- MongoDB (Mongoose)
- JWT Authentication
- Nodemailer
- GROQ AI
